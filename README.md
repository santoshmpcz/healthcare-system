Healthcare Appointment Booking System
Tech Stack: Spring Boot, Spring Data JPA, Thymeleaf, Bootstrap, jQuery, MySQL, Maven, Git
Project Description:
Developed a full-stack healthcare web application to automate appointment scheduling, patient management, and hospital workflows. The system enables patients to book appointments, doctors to manage schedules, and admins to control operations with real-time dashboards. Designed using MVC architecture for scalability, performance, and maintainability.
Key Responsibilities & Achievements:
•	Engineered a role-based secure system (Admin, Doctor, Patient, Staff) ensuring controlled access and data integrity 
•	Developed end-to-end appointment workflow (registration → booking → approval → billing → cancellation) 
•	Built RESTful backend services and integrated with frontend using Thymeleaf 
•	Designed responsive UI for login, dashboard, booking, and reporting modules 
•	Implemented dynamic dashboards for real-time tracking of appointments and users 
•	Developed slot management system with status handling (Pending, Approved, Rejected, Cancelled) 
•	Integrated email notifications for user communication using Java Mail API 
•	Built file upload/download module for hospital documents
