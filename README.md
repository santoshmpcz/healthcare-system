Healthcare Appointment Booking System

Tech Stack: Spring Boot, Spring MVC, JPA (Hibernate), Thymeleaf, Bootstrap, MySQL

Description:
Developed a full-stack healthcare web application to manage patient appointments, doctor schedules, and hospital workflows with real-time dashboards and secure role-based access. The application follows Spring MVC architecture with Thymeleaf for server-side rendering, ensuring seamless integration between backend logic and frontend UI.

Key Contributions:

Built an end-to-end appointment management system (registration → booking → approval → billing)
Implemented role-based authentication and authorization (Admin, Doctor, Patient, Staff)
Developed the application using Spring MVC architecture with Thymeleaf-based server-side rendering
Designed responsive UI components for dashboard, appointment booking, and reports using Bootstrap
Created dynamic dashboards for real-time tracking of appointments and system activities
Implemented slot management system with multiple status handling (Booked, Cancelled, Completed, etc.)
Integrated email notifications using Java Mail API for appointment updates
Developed file upload and download functionality for managing documents
Applied input validation, exception handling, logging (Log4j), and unit testing (JUnit)
